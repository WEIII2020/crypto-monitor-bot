# 🌐 腾讯云 Web 终端上传指南

## 方法 1: 使用腾讯云控制台的文件上传功能

### 步骤：

1. **登录腾讯云控制台**
   - 访问: https://console.cloud.tencent.com/cvm
   - 找到你的服务器（119.28.43.237）

2. **打开 Web 终端**
   - 点击服务器右侧的 "登录"
   - 选择 "标准登录方式"
   - 会打开一个网页版终端

3. **上传文件**
   - 在 Web 终端窗口，找到顶部工具栏
   - 应该有一个 "上传文件" 或 "文件传输" 按钮 📁
   - 点击后选择本地文件: `/tmp/crypto-bot-v2.0.tar.gz`
   - 上传到服务器的 `/root/` 目录

4. **验证**
   - 在终端执行: `ls -lh /root/crypto-bot-v2.0.tar.gz`
   - 应该看到 410K 的文件

## 方法 2: 使用 rz 命令（如果支持）

如果 Web 终端支持 lrzsz：

1. SSH 登录后执行:
```bash
cd /root
rz
```

2. 选择本地的 `/tmp/crypto-bot-v2.0.tar.gz`
3. 等待上传完成
