# 📤 文件上传指南

## 文件信息
- **本地路径**: `/tmp/crypto-bot-v2.0.tar.gz`
- **文件大小**: 410K
- **目标位置**: `root@119.28.43.237:/root/`

---

## 🎯 方案 1: 使用 Transmit（推荐 - Mac 最佳）

### 下载 Transmit
- 官网: https://panic.com/transmit/
- 或 App Store 搜索 "Transmit"

### 配置连接
1. 打开 Transmit
2. 点击 "+" 新建连接
3. 填入信息：
   - **协议**: SFTP
   - **地址**: `119.28.43.237`
   - **用户名**: `root`
   - **密码**: （你的服务器密码）
   - **端口**: `22`
4. 点击"连接"

### 上传文件
1. 在 Finder 中打开 `/tmp/` 目录（按 Cmd+Shift+G）
2. 找到 `crypto-bot-v2.0.tar.gz`
3. 拖拽到 Transmit 的右侧窗口（服务器的 `/root/` 目录）
4. 等待上传完成（几秒钟）

---

## 🎯 方案 2: 使用 Cyberduck（免费）

### 下载 Cyberduck
- 官网: https://cyberduck.io/
- 免费开源

### 配置连接
1. 打开 Cyberduck
2. 点击"新建连接"
3. 选择协议: **SFTP (SSH File Transfer Protocol)**
4. 填入：
   - **服务器**: `119.28.43.237`
   - **端口**: `22`
   - **用户名**: `root`
   - **密码**: （你的服务器密码）
5. 点击"连接"

### 上传文件
1. 连接成功后，在 Cyberduck 中导航到 `/root/`
2. 点击工具栏的"上传"按钮
3. 选择 `/tmp/crypto-bot-v2.0.tar.gz`
4. 点击"上传"

---

## 🎯 方案 3: 命令行（需要正确密码）

### 3.1 确认密码
先测试 SSH 登录：
```bash
ssh root@119.28.43.237
```

如果能登录，说明密码正确，然后退出（输入 `exit`）。

### 3.2 上传文件
```bash
scp /tmp/crypto-bot-v2.0.tar.gz root@119.28.43.237:/root/
```

## 🔐 如果忘记密码

### 腾讯云控制台重置密码
1. 登录腾讯云控制台: https://console.cloud.tencent.com/
2. 进入"云服务器"页面
3. 找到你的服务器（119.28.43.237）
4. 点击"更多" → "密码/密钥" → "重置密码"
5. 设置新密码
6. 重启服务器使密码生效
