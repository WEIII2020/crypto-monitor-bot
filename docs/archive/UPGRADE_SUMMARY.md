## 🐋 庄家行为检测（已实现）

### 5 种检测模式

| 信号 | 条件 | 含义 |
|-----|------|------|
| 🟢 吸筹 | 价格横盘 ±3% + 交易量 5x | 庄家建仓，压低价格吸筹 |
| 🔴 出货 | 价格上涨 10%+ + 交易量 5x | 庄家拉升出货，散户接盘 |
| ⚠️ 假突破 | 价格上涨 8%+ + 交易量 < 2x | 缩量上涨，可能是假突破 |
| 💀 恐慌 | 价格下跌 10%+ + 交易量 5x | 恐慌性抛售 |
| 📊 对敲 | 交易量 10x + 价格波动小 | 可能对敲制造成交量 |

### 告警示例

```
🟢 放量不涨，疑似吸筹
📈 ENJ/USDT
💰 价格: $0.2345
📊 涨跌: +1.2%
🔊 交易量: 6.3x 平均值
```

## ⚠️ 注意事项

### 系统限制
```
1. 检测延迟：30秒（每30秒检查一次）
2. 历史窗口：5分钟（基于最近5分钟数据）
3. 告警冷却：5分钟（同一信号5分钟内不重复）
4. 订阅上限：1024个币种（Binance限制）
```

### 风险提示
```
⚠️ 这是辅助工具，不是交易信号
⚠️ 告警不保证准确，需要人工判断
```
